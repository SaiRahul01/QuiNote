# Quinote 
# Secure Place for all Your  Notes

### **Description**
Quinote(Quick~Note) is Full Stack Web app made using the MERN Stack

### Technologies used:
1. ReactJS
2. ReactBootstrap
3. NodeJs 
4. ExpressJs
5. MongoDb

### Functionalities
1. Register and Login using Email and Password.
2. CRUD (create, read, update and Delete) Operations on Notes.
